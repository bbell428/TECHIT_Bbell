# 앱 이름
Firebase FireStore를 활용한 게시판

# 주요기능
- 게시글 생성/수정/삭제 기능
- 댓글/하트 달기 기능
- Firebase 연동 및 Appstorage로 인한 로컬 데이터 유지

# 작동환경
- Xcode 15.4
- iOS 17.5
