# 씩스키오스크
6조의 키오스크!!

# 주요기능
- 매장, 포장 선택
- 음식 종류별 필터링, 장바구니 담기,
- 장바구니 아이템 추가 삭제, 총 금액
- 결제 방법 선택

# 작동환경
- Xcode: 15.4
- iPad: 17.2
